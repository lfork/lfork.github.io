# 启发规则
+ 改进软件结构提高模块独立性
+ 模块规模应该适中
+ 深度、宽度、扇入扇出都应该适当
+ 模块的作用域应该在控制域之内
+ 力争降低模块接口的复杂度
+ 设计单入口单出口的模块
+ 模块功能应该可以预测
